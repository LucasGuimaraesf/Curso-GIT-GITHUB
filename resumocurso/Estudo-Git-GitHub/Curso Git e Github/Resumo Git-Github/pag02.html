Introdução GitHub

O que é Github?

  • Repositório de códigos que surgiu em 2008.
  • Comitar/Comit : é quando você salva o arquivo no repositório local.
  • Push: o seu arquivo será enviado para o repositório remoto.
  • O GitHub é um dos mais famosos REPOSITÓRIOS REMOTOS que existem.
  • Além de repositório é uma "rede social dos desenvolvedores". -Gustavo Guanabara

    • Coisas que o GitHub faz:

    1. Repositórios ilimitados
    2. Hospedagem de código-fonte (qualquer linguagem de programação)
    3. Características de rede social
    4. GitHub Pages integrado
    5. Colaboração
    6. Forks de um projeto e continuar por conta própria, desde que respeite a licença
    7. Em 2008 foi comprada pela Microsoft
